# План реализации: prompt-engineering-demo

## Обзор

Реализация самодостаточного CLI-скрипта `prompt_engineering_demo.py` на Python.
Скрипт демонстрирует разницу между базовым и улучшенным (few-shot) промптами,
отправляет оба через `openai_client.chat()`, сравнивает ответы и генерирует `artifact_README.md`.

## Примечания

- Задачи с `*` опциональны и могут быть пропущены для быстрого MVP
- Каждая задача ссылается на конкретные требования для трассируемости
- Property-тесты используют библиотеку **Hypothesis** (`@settings(max_examples=100)`)
- Unit-тесты покрывают граничные случаи: отсутствие API-ключа, исключение от `chat()`, невалидный JSON
- Скрипт не импортирует `prompts.json` и не зависит от `ai_direct.py`
